
Project : InOrder Simulator

1. Unzip the package. Place the input file you want to test
	in the same folder as that of the java files
	
2. Get inside the folder which contains java files, makefile and the input file

3. Now type 'make' to build and run the project

4. While issuing 'loadfile <inputfile>' command, give the input file name,
   which you have placed in the folder.
   
5. type 'make' to run the project everytime
